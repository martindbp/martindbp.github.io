Are parents picking more unique names for their children?

2019-10-23 22:03

Comments

I feel like it's a topic that often comes up talking to other parents. Instead of relying on a gut feeling or anectdata, let's check!

Being Swedish, of course I'm mostly interested in Swedish name statistics, and luckily SCB (the central statistics bureau) provides an Excel spread sheet with the frequency statistics of the top 100 names for boys and girls from 1998 to 2017. Let's download it:

! wget -O stats.xlsx https://www.scb.se/hitta-statistik/statistik-efter-amne/befolkning/amnesovergripande-statistik/namnstatistik/pong/tabell-och-diagram/nyfodda--efter-namngivningsar-och-tilltalsnamn-topp-100/Namn-1998-/

Now we can read the spreadsheet with Python and pandas/numpy (I bet it would be easier to just do this in Excel, but I'm not an Excel-ninja...) and collect the statistics.

The question is what's a good measure for the unevenness of the name distribution. I'm sure there are many options for this, but I'll pick perplexity which is based on entropy. The entropy of the distribution measures the disorder of the distribution, i.e. we get the highest entropy if all names have an equal frequency, and low entropy if everyone named their child the same way. Perplexity can be seen as the average number of "choices" one has in the random variable, e.g. a 6-sided fair die has a perplexity of 6, while a a die that always ends up on one number has a perplexity of 1. This makes it a bit easier to interpret. Let's load the Excel file and run the calculations:

import pandas as pd
import numpy as np

START_YEAR, END_YEAR = 1998, 2017
FEMALE, MALE = 0, 1
sex_perplexity = ([], [])
with open('stats.xlsx', 'rb') as f:
    for year in range(START_YEAR, END_YEAR+1):
        for sex in [FEMALE, MALE]:
            sheet_idx = 2*(year - START_YEAR) + sex + 1
            df = pd.read_excel(f, sheet_name=sheet_idx, header=6, usecols='B:D')
            col_name = 'Antal' if 'Antal' in df.columns else 'Antal bärare'
            freqs = df[col_name].values[1:].astype(float)
            probs = freqs / freqs.sum()
            entropy = -(probs * np.log(probs)).sum()
            perplexity = np.exp(entropy)
            sex_perplexity[sex].append(perplexity)

Now we can plot the perplexity over time and see how it looks:

show code (8 lines) hide code 
import seaborn as sns
import matplotlib.pyplot as plt
n_years = END_YEAR-START_YEAR+1
data = pd.DataFrame(data={'year': 2*list(range(START_YEAR, END_YEAR+1)),
                          'sex': n_years*['female']+n_years*['male'],
                          'perplexity': sex_perplexity[FEMALE]+sex_perplexity[MALE]})
fig, ax = plt.subplots(figsize=(10, 5))
ax = sns.lineplot(x='year', y='perplexity', hue='sex', data=data)

It seems like within the top 100 names, the answer is an unequivocal yes! Parents are steadily spreading out their name choices more and more, approaching 90/100 which is quite equal.
